## рҹҡҖ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Ensure you have Python installed on your system. You may also need a C++ compiler installed for `dlib` to build correctly on some operating systems.
*   [Python 3.x](https://www.python.org/downloads/)
*   Git

### Installation

1.  **Clone the repository:**
    
    ```bash
    git clone [https://github.com/halukcansarioz/Face-Recognition.git](https://github.com/halukcansarioz/Face-Recognition.git)
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd Face-Recognition
    ```

3.  **Create a virtual environment (Recommended):**

    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows use: venv\Scripts\activate
    ```

4.  **Install dependencies:**
    *(Ensure you have a `requirements.txt` file, or install the main libraries manually like `pip install opencv-python face_recognition numpy`)*

    ```bash
    pip install -r requirements.txt
    ```

5.  **Run the application:**

    ```bash
    python main.py
    ```
    *(Note: Change `main.py` to the actual name of your starting script, e.g., `recognize.py` or `app.py`)*
